invariant.core.count

count

(count {:keys [name sources predicate]} count-predicate & args)

An invariant on the number of elements matching a given specter navigator and predicate.

For example, to ensure that there are at most 5 queries within a vector of operations.

(invariant/count
  {:name    :validator/at-most-five-queries
   :sources [:operations ALL #(= (:type %) :query)]}
  <= 5)
